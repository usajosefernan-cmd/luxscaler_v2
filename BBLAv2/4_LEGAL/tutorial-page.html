üõ†Ô∏è Common Issues & Solutions

‚ùå Face looks different in the 4K output

Why: Creativity was set too high (>5) or AI emphasized facial features

Solution:

  • Use Creativity 2-4 for portraits
  • Choose preview that matches your face best
  • Use "Balanced Polish" preset (conservative style)

‚ùå Image is too blurry/soft

Why: Fractality was set too low (<4) or original image was very low resolution

Solution:

  • Increase Fractality to 7-8
  • Use "Max Sharpness" preview
  • Try "Forensic Recovery" if original is damaged

‚ùå Colors look oversaturated/fake

Why: Color Boost was set too high (>+4) or image was already color-graded

Solution:

  • Reduce Color Boost to +2 or 0
  • Use "Balanced Polish" (conservative color)
  • Use "Trend" if you want artistic colors

‚ùå Lighting looks weird/artificial

Why: Light Boost was set too extreme (>+4 or <-4) or original lighting was inconsistent

Solution:

  • Keep Light Boost between -2 and +3
  • Use "Studio Lighting" preview (professionally balanced)
  • Choose a preview that matches your intended style

‚ùå Text/details look AI-generated

Why: Creativity was too high (AI "imagined" details) or image type is complex (mixed text + photo)

Solution:

  • Set Creativity to 0-1 for documents
  • Use "Forensic Recovery" (safe, minimal changes)
  • Avoid "Creative Reality" or "Max Sharpness"

üí° Advanced Tips

Tip 1: Layer Your Settings

Don't max out every slider. Instead:

  1. Set Creativity based on image type (low for portraits, high for landscapes)
  2. Set Fractality based on desired detail level
  3. Only adjust Light/Color Boost if original needs help

GOOD Creativity 4, Fractality 7, Light +1 (balanced)

BAD Creativity 10, Fractality 10, Light +5, Color +5 (overdone)

Tip 2: Preview Tells You Everything

All 6 variations use the same base settings but different "creative directions."

  • If preview #1 (Forensic) looks best ‚Üí Image has damage to fix
  • If preview #3 (Trend) looks best ‚Üí Image benefits from artistic style
  • If preview #6 (Max Sharpness) looks best ‚Üí Image needs maximum detail

Tip 3: Save Your Settings

If you find settings you like (e.g., "Creativity 4, Fractality 7, Light +2"), write them down and use them for similar images.

Tip 4: Use Batch Mode for Consistency

If you're upscaling multiple photos (e.g., product catalog, photo shoot):

  1. Use the same settings for all images
  2. Ensures consistent style across your collection
  3. Professional results for galleries/portfolios
